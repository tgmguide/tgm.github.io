TETRIS TGM SETUP GUIDE
A handy guide on how to set up TGM and Texmaster

[Non-TL;DR Guides to follow!]
TL;DR BELOW
TGM1/TAP
Install shmupmame first: [download]
Then paste mametgm into the shmupmame folder: [download]

Now you'll need roms. This guide cannot hook you up with those due to legal reasons but the filenames you're looking for are as follows:
TGM1: "tgmj.zip" and "cpzn2.zip". If you decide to not use shmupmametgm for whatever reason and want to run the regular newest MAME version for whatever reason, you'll also need "qsound.zip".
TGM2/TAP: "tgm2.zip" and "tgm2p.zip". Keep in mind you MUST start "tgm2p.zip" if you want to play the TAP exclusive modes such as T.A. Death.
By the way, I heard that there may be a website which declares itself as a paradise which has these. ;)
To change key layout, press Tab and navigate with WASD and H to enter menus. Then set the keys to whatever you like.

TGM3 Terror Instinct
First you'll need to go on a small scavenger hunt for the actual game files. Doing a google search for it should usually do the job.
Now, you absolutely MUST download the altinput package unless you want stupid amounts of input lag: [download]
To configure your keyboard layout, you need to open the cfg that comes with altinput and check out the link to a Microsoft page that's in there. Then do some ctrl-F searching for the keys you need and paste the according codes into the cfg file.
When starting the game, start it using tgm3_launcher.exe! Don't use the normal launcher!
If you've done everything correctly then you too can now enjoy all of these games!

Texmaster
But WAIT! I don't like piracy, it's unethical! HOW DO I CIRCUMVENT THIS?
Well, lucky you! We have Texmaster! A free clone with which you can play all TGM modes (aside from Sakura from TGM3)!
Note: for all downloads in this section, you need 7-Zip! Go google it up or something.
Here is the download link to the latest Texmaster version: [download]
But man, regular Texmaster looks like shit. Let's fix this, shall we?
Grab this "modpack" called Termino Velocity! It absolutely kicks vanilla Texmaster's ass with an entirely new look and feel to it. Download's here: [download]
In order to change your key layout, open the .ini and then the "keys.html" file in the "html" folder to see which numbers you need to use for the individual keys. Don't forget to set Player 1's key layout to your custom one.

And there you go. A quick TL;DR (which is still fairly long) to just get the games up and running.
Now go out there and stack to your hearts content!
